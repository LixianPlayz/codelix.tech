Python Coding Tutorials

1. Printing Text

print("Hello, world!")

This prints text to the console. Great for checking if your code runs correctly!

2. Loops

for i in range(5):
    print("Number:", i)

This loop runs five times, printing the numbers 0 through 4.

3. Functions

def greet(name):
    print("Hello, " + name + "!")

greet("Charlie")

Functions help you reuse code. This one greets the person whose name you pass in.

LuaU Coding Tutorials

1. Printing a Message

print("Hello, Roblox!")

This prints text to the Roblox output console. It’s a simple way to test scripts.

2. Moving a Part

local part = game.Workspace.Part
part.Position = Vector3.new(0, 10, 0)

This script moves a part to a new position in the workspace.

3. Connecting a Function

local part = game.Workspace.Part
part.Touched:Connect(function(hit)
    print(hit.Name .. " touched the part!")
end)

This connects a function that runs when the part is touched, printing what touched it.
