Beginner Python Tutorials

Three concise beginner-friendly tutorials with examples. Copy the snippets and try them locally.

1. Hello, Python — Your First Script

Writing and running a Python script. This example prints a greeting and demonstrates basic variables and comments.


# hello.py
name = "Lix"
greeting = f"Hello, {name}! Welcome to Python."
print(greeting)

How to run: Save as hello.py and run python hello.py in your terminal.

2. Control Flow — Ifs and Loops

Use conditionals and loops to control program flow. This snippet prints numbers and indicates whether they are even or odd.


# even_odd.py
for i in range(1, 7):
    if i % 2 == 0:
        print(i, "is even")
    else:
        print(i, "is odd")

Tip: Try changing the range to see more results and experiment with while loops.

3. Functions & Simple I/O

Functions let you package logic. This example shows a simple function that computes square and uses input/output.


# square.py
def square(x):
    return x * x

try:
    val = float(input("Enter a number: "))
    print("Square:", square(val))
except ValueError:
    print("Please enter a valid number.")

Experiment: Modify square to return the cube or create a new function for factorial.
