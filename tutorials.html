Working with Lists & Loops

Create lists, iterate with loops, and perform simple operations.

Lists are ordered collections. Loops help you process items efficiently.

# lists_loops.py
fruits = ["apple", "banana", "cherry"]

for i, fruit in enumerate(fruits, start=1):
    print(f"{i}. {fruit}")

# list comprehensions
upper_fruits = [f.upper() for f in fruits]

Reading & Writing Files

Open, read, and write text files safely with context managers.

File I/O is essential for saving program state, logs, and data.

# file_io.py
# write
with open("notes.txt", "w", encoding="utf-8") as f:
    f.write("Remember: Python is fun!\n")

# read
with open("notes.txt", "r", encoding="utf-8") as f:
    content = f.read()
print(content)
  • Use open() with context managers (with)
  • Read and write files using text mode
  • Handle text encoding safely
